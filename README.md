# flask-celery-redis

flask-celery-redis for asynchrony task

## Installation

- Créer le venv `python3 -m venv venv`
- Activer le venv `. venv/bin/activate`
- Installer les dépendances `pip install -r requirements.txt`

- Lancer l'application
    - Dans l'env de Dév (avec le serveur de développement) : `flask --app app.py run` ($env:FLASK_APP = "app.py" then flask run)
